# すき焼き (2 人分)

## 材料

- 創味のつゆ: 50 cc
- 水: 50 cc
- 酒: 適量
- 砂糖: 小さじ 2
- 牛肉: 200 g
- 野菜: 400 g くらい
- 糸こんにゃく: 100 g

## 手順

```
def prepare():
    創味のつゆを水で割る

    if type(糸こんにゃく) != アク抜き不要:
        糸こんにゃくを 5 分間茹でる


def main():
    prepare()

    鍋を熱する
    油をひく
    牛肉を鍋に入れる
    牛肉の上に砂糖をふりかける

    while True:
        if 火の通り具合(牛肉) != 半分:
            break

    つゆを入れる
    野菜を入れる
    糸こんにゃくを牛肉に触れないように入れる

    for _ in range(10):
        混ぜる
        sleep(60)


if __name__ == '__main__':
    main()
```
